# CaptureTool

Windows向けのスクリーンキャプチャツールです。ホットキーによるキャプチャ、柔軟なファイル命名、オーバーレイ通知などの機能を備えています。

---

## 動作環境

- Windows 7 / 8 / 8.1 / 10
- .NET Framework 4.6

---

## 主な機能

### キャプチャ機能
- **ウィンドウキャプチャ** — フォアグラウンドウィンドウをキャプチャ
- **スクリーンキャプチャ** — アクティブな画面全体をキャプチャ
- **クリップボード取得** — クリップボードのテキストをファイル名欄に貼り付け
- **キャプチャ方式の切り替え** — GDI (Vista以前) / 通常 / GDI / DesktopDuplication の4種類

### ホットキー
3種類のホットキーを独立して設定できます（修飾キー + キーの組み合わせ）。

| 操作 | デフォルト |
|------|-----------|
| ウィンドウキャプチャ | Ctrl + Q |
| スクリーンキャプチャ | Alt + Q |
| クリップボード取得 | Alt + S |

タブごとに異なるホットキーを設定可能です。

### ファイル命名
- 連番付加（桁数指定・カウントのリセット／続行）
- 接続語の指定（例: `Capture_001.png` の `_` 部分）
- ファイル名候補リストからの選択

#### ファイル名オプション（動的置換）
| プレースホルダー | 内容 |
|----------------|------|
| `<WindowTitle>` | キャプチャ対象ウィンドウのタイトル |
| `<Date>` | キャプチャ日付（yyyyMMdd） |
| `<Time>` | キャプチャ時刻（HHmmss） |
| `<Dir>` | 保存フォルダ名 |
| `<HostName>` | コンピューター名 |

### 保存形式
- **PNG** / **JPG** を選択可能
- ピクセル形式: 32bppArgb / 24bppRgb / 16bppRgb555 / 8bppIndexed

#### PNG圧縮
| 方式 | 説明 |
|------|------|
| 無し | 圧縮なし |
| Optipng | optipng.exe を使用（レベル1〜8） |
| Zopfli | zopflipng.exe を使用（レベル1〜7） |

圧縮には外部ツール（`optipng.exe` / `zopflipng.exe`）が必要です。`CallCompress.exe` 経由で呼び出されます。

### オーバーレイ表示
キャプチャ後にキャプチャ画像とファイル名を画面上に一時表示します。
- 表示時間（ミリ秒）
- 表示位置（左上 / 右上 / 左下 / 右下）
- 表示サイズ（幅・高さ）
- タブ番号・ファイル名の表示有無

### 保存先設定
- フォルダパスの直接入力・ダイアログ参照
- お気に入りフォルダの登録・適用・名前変更・削除
- 親フォルダへの移動ボタン
- エクスプローラーで保存先を開くボタン

### その他
- **タスクトレイアイコン化** — 最小化時にタスクトレイへ格納
- **終了時設定保存** — アプリ終了時に自動で設定を保存
- **スクリーンキャプチャ時ウィンドウ非表示** — キャプチャ中はCaptureTool本体を非表示にする
- **常に最前面に表示**
- **Aero対応** — DWM拡張フレーム境界を使った正確なウィンドウ領域取得
- **マウスカーソルのキャプチャ**（カーソルアイコンを矢印に固定するオプションあり）
- **IPアドレス → ホスト名変換** — hostsファイルを参照してウィンドウタイトル内のIPをホスト名に置換
- **ウィンドウタイトル置換設定** — 正規表現対応の置換ルールを複数登録可能
- **キャプチャ時にファイル名を設定** — キャプチャ実行のたびに保存先をダイアログで指定

## タブ機能

複数のタブを作成し、それぞれ独立した設定（ホットキー・保存先・ファイル名など）を持つことができます。タブは別ウィンドウとして切り出すことも可能です。

---

## UI設定

設定ウィンドウのメイン画面に表示する項目を個別にオン／オフできます。

- キー設定
- キー設定表示（現在のホットキー一覧）
- 保存形式設定
- 保存先設定
- オーバーレイ設定
- キャプチャ設定
- その他設定

---

## 設定の保存

設定はアプリケーションディレクトリ内の XML ファイルに保存されます。

| ファイル名 | 対象 |
|-----------|------|
| `setting.xml` | タブ1の設定 |
| `setting1.xml`, `setting2.xml`, ... | 追加タブの設定 |
| `DefaultSetting.xml` | デフォルト設定 |

## プロジェクト構成

```
CaptureTool.sln
├── CaptureTool/          # メインアプリケーション（WPF）
│   ├── MainWindow.xaml          # メインウィンドウ
│   ├── MainInstance.xaml        # タブごとのインスタンス
│   ├── Settings.cs              # 設定モデル（XML読み書き）
│   ├── MainProcess.cs           # キャプチャ処理本体
│   ├── HotKeySettings.cs        # ホットキー管理
│   ├── HotKey.cs                # Win32 ホットキー登録
│   ├── OverlayWindow.xaml       # オーバーレイウィンドウ
│   ├── OverlayDialog.xaml       # 通知ダイアログ
│   ├── MiniWindow.xaml          # ミニウィンドウ
│   ├── TabWindow.xaml           # タブの別ウィンドウ表示
│   ├── NotifyIconWrapper.cs     # タスクトレイアイコン
│   ├── Logger.cs                # ログ出力
│   └── Pages/                   # 設定UI（UserControl）
│       ├── KeyBind.xaml
│       ├── SaveDir.xaml
│       ├── SaveFormat.xaml
│       ├── Overlay.xaml
│       ├── CaptureSetting.xaml
│       ├── Other.xaml
│       └── UISetting.xaml
└── CallCompress/         # 圧縮ツール呼び出し用の補助EXE
    └── Program.cs
```
